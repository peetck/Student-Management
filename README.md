## About The Project

<img src="https://github.com/peetck/Student-Management/blob/master/images/main.png?raw=true" alt="main">

พวกเราได้ทำโปรแกรมที่จะช่วยให้อาจารย์หรือผู้ใช้งานมีความสะดวกยิ่งขี้นในการจัดการเกี่ยวกับข้อมูลนักเรียนในด้านต่างๆ เช่น รายชื่อ ข้อมูลส่วนตัว หรือ คะแนนวิชาต่างๆ ได้อย่างง่ายดายมากขึ้น

โปรแกรมนี้เป็นส่วนหนึ่งของวิชา OBJECT ORIENTED PROGRAMMING (1/2019) KMITL

ฟังก์ชั่นหลักๆของโปรแกรม:
* จัดการข้อมูลนักเรียน
* จัดการคะแนน
* แสดงผลคะแนนออกมาเป็นกราฟ
* ระบบล็อกอิน

ขอขอบคุณ:
* <a href='https://github.com/LGoodDatePicker/LGoodDatePicker'>LGoodDatePicker</a>
* <a href='http://www.jfree.org/jfreechart/'>JFreeChart</a>
* <a href='https://github.com/atarw/material-ui-swing'>Material UI Swing</a>
* <a href='https://poi.apache.org/'>Apache POI</a>
* <a href='https://www.flaticon.com/'>Flaticon</a>
